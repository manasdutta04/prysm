# Prysm - AI-Driven Customer Feedback Analyzer

## AI-Driven Customer Feedback Analyzer
A tool that ingests customer feedback from multiple sources (support tickets,
reviews, social media comments) → AI clusters them into topics and autogenerates actionable insights.

## Features:
- AI-driven sentiment analysis & trend detection.
- “Emerging issues” alerts before they blow up.
- Department-specific reports (Support, Product, Marketing).
- Slack & email notifications.
- Unique Angle: Goes beyond sentiment → actually tells teams what to fix.
- Target Users: Product managers, customer success teams.
